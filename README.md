## ğŸ§  Overview

**AyurSutra** is a centralized, digital platform designed to modernize **Panchakarma** and Ayurvedic therapy management.  
It replaces fragmented, paper-based workflows with a unified system for:

- Patient data management  
- Intelligent therapy scheduling  
- Progress tracking and visualization  
- Notifications & engagement  

The goal is to streamline operations for Ayurvedic clinics and wellness centers while giving patients a transparent and engaging wellness journey.   

---

## âœ¨ Key Features

### 1. Centralized Management
- Secure, unified storage of patient records and therapy details  
- Eliminates manual, paper-based documentation  
- Supports both small clinics and large wellness centers  

### 2. Automated Scheduling
- Custom scheduling algorithm to optimize therapy sessions  
- Considers **therapist availability** and session constraints  
- Calendar-based planning with easy modification and rescheduling  

### 3. Pre / Post Procedure Notifications
- Automated reminders via **app, SMS, or email**

### 4. Real-Time Tracking & Visualization
- Visual timelines, charts, and progress bars for therapy and recovery  
- Enables practitioners to monitor patient journeys at a glance  
- Helps standardize therapy quality and outcomes   

### 5. Future Enhancements (Planned)
- Advanced patient visualization tools  
- Automated feedback and follow-up loops  
- Payment gateway integration for seamless billing

## ğŸ§© System Architecture (High-Level)
  

**Flow Summary:**

1. **Patient Dashboard**
   - Patient enters health details and therapy-related information.
   - Data is submitted to the backend via an API call.

2. **Backend Processing (AyurSutra)**
   - Stores and queries data from MongoDB.
   - Runs the **scheduling algorithm** for therapy planning.
   - Assigns practitioners based on availability and requirements.
   - Optionally uses LLM processing for extracting and structuring unstructured data (future scope).

3. **Practitioner Dashboard**
   - Practitioners retrieve assigned patient data.
   - View therapy plans, schedules, and progress.
   - Update therapy status and notes in real time.
